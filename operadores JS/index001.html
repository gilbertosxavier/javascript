Operadores em JavaScript

  • Aritméticos

    • O sinal '+' representa Adição

    • O sinal '-' representa Subtração

    • O sinal '*' representa Multiplicação

    • O sinal '/' representa divisão com números reais, com vírgula.

    • O sinal '%' representa divisão inteira, sem vírgula.

    • O sinal '**' representa Potênciação

    • O sinal '( )' representa Parênteses

    • Operador de Atribuição é representado pelo caractere '='.
      Seguem Exemplos:

      • const nome = Gilberto
      • let idade = 33
      • var cidade = Pelotas

  • Relacionais

    • Operadores relacionais, como o próprio nome diz estabelecem uma relação entre algo.

    Ainda dentro dos operadores relacionais, temos a Operação de identidade que é representada pelo simbolo '===' onde ele verifica se as duas coisas que estão sendo comparadas são identicas.
    Vamos ao exemplo:

    • 5 == 5 -> true
      Aqui o JavaScript entende que é igual;

    • 5 == '5'-> true
      Aqui o JavaScript entende que é igual, embora sejam tipos diferentes primeiro um número e o segundo uma string;

    • 5 === '5' -> false
      Aqui o JavaScript entende que o valor é diferente, pois com o comparador '===' ele verifica se são identicos.

    • 5 === 5 -> true
      Aqui o JavaScript entende que o valor é igual, pois são identicos.

  • Lógicos

    • No JavaScript temos 3 Operadores Lógicos que são: Negação ' ! ', Conjunção ' && ' e Disjunção ' || '
    Podemos verificar que a negação muda o valor de sua saída, Conjunção só será true se todas as váriaveis de entrada forem satisfeitas e a Disjunção será true se pelo menos uma das variáveis de entrada for satisfeita.

    1. Negação '!':
      !true -> false
      !false -> true

    2. Conjunção '&&':
      true && true -> true
      true && false -> false
      false && true -> false
      false && false -> false

    3. Disjunção '||':
      true && true -> true
      true && false -> true
      false && true -> true
      false && false -> false

  • Ternário

    Operador ternário consiste em uma operação onde o resultado será true ou false dependendo do teste feito. Possui essa estrutura que veremos a seguir.
    teste ? true : false
    Exemplos:

    • var nota = 8;
      nota >= 7 ? 'APROVADO' : 'REPROVADO';
    • var x = 8;
      var res = x % 2 == 0 ? 'SIM' : 'NÃO'
      res == SIM

Precedência de Operadores

Sempre devemos respeitar essa ordem nas operações, pois o JavaScript se baseia nela.

  1. Aritméticos

    Sendo as operações de multiplicação, divisão, adição e subtração devem ser executadas pela ordem de aparição, sendo da esquerda para direita.

    1. Os Parênteses '( )' tem preferência;
    2. A Potenciação '**' vem logo na sequência;
    3. Após temos multiplicação, divisão sem e com resto '*' , '/' , '%' ;
    4. Por fim temos adição e subtração '+', '-'/

  2. Relacionais

    Os operadores relacionais não tem ordem entre si, porém devem ser resolvidos em da esquerda para direita.

    • O simbolo '>' representa a expressão 'maior que';

    • O simbolo '<' representa a expressão 'menor que';

    • O simbolo '>=' representa a expressão 'maior ou igual a';

    • O simbolo '<=' representa a expressão 'menor ou igual a';

    • O simbolo '==' representa a expressão 'igual a';

    • O simbolo '!=' representa a expressão 'diferente';

  3. Lógicos

    Se houverem várias condições na mesma sentença a ordem de execução é SEMPRE Negação,Conjunção e Disjunção.

    1. Negação: '!';

    2. Conjunção: '&&';

    3. Disjunção: '||';

  4. Ternário

    Sempre será o último executado, pois indica uma condição de resultado.
    teste ? true : false
